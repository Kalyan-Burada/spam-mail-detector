# Spam Detection Project

This project is a machine learning model that detects spam messages.

## Files
- spam.csv – Dataset
- model.pkl – Trained model
- vectorizer.pkl – Text vectorizer

## How to Use
1. Load model
2. Transform text
3. Predict spam/ham

## Tech Stack
- Python
- Scikit-learn
- Pandas
